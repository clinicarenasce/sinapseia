Dúvidas frequentes

Funciona sem internet? ▼

A transcrição com Whisper é 100% local — funciona offline. A formatação com IA usa a API Groq e precisa de internet só nesse passo.

Meus áudios ficam armazenados na nuvem? ▼

Não. O modelo Whisper roda no seu PC. Nenhum áudio é enviado a servidores externos. Só o texto transcrito é enviado à API Groq para formatação.

Posso cancelar o plano Pro? ▼

Sim, a qualquer momento. Após o cancelamento você continua com acesso Pro até o fim do período pago. Sem multas.

Como recebo minha licença Pro? ▼

Automaticamente por email logo após o pagamento. Uma chave no formato SINAPSE-XXXX-XXXX-XXXX é gerada e enviada. Basta colar no app.

Precisa de configuração técnica? ▼

Precisa de Python 3.10+ e Chrome instalados. Na primeira execução o modelo Whisper é baixado automaticamente (~800MB). Depois é só usar.
